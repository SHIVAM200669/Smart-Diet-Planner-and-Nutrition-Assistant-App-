
Food Diet Planner - Java Swing + MySQL Project (Modern UI)

1. Setup MySQL database 'food_diet_planner' and 'users' table.
2. Add MySQL connector jar in lib folder.
3. Compile: javac -cp .;lib/mysql-connector-java-8.1.1.jar src/*.java
4. Run: java -cp .;lib/mysql-connector-java-8.1.1.jar src.Main
5. Modern UI with Signup and Login pages included.
